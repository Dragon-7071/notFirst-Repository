1. Потрібно натиснути в меню кнопку +, потім New repository

2. Репозиторій потрібен для збереження всіх змін виконуваного проекту.

3. Натиснути на кнопку з назвою main або master потім вказати назву гілки та натиснути Enter

4. Потрібно зробити якісь зміни в новій гілкі потім натиснути Create pull request потім Merge pull request.

5. Потрібно натиснути на кнопку Code потім Download ZIP.
